# hassh

یک پروژهٔ سادهٔ PHP (بات / وب‌اپ).

## توضیح کوتاه

محتوای اصلی در پوشهٔ `@@###@aahash/` قرار دارد؛ شامل `bot.php`، `config.php` و پوشهٔ `lib/`.

## نیازمندی‌ها

- PHP 7.4+
- وب‌سرور (Apache/Nginx) یا PHP built-in

## اجرا (سریع)

1. کلون ریپو:
```bash
git clone https://github.com/mamlh1efootball-bit/hassh.git
cd hassh
```
2. فایل پیکربندی را ویرایش کن: `@@###@aahash/config.php` (توکن‌ها و تنظیمات DB).
3. اجرای محلی:
```bash
php -S localhost:8000
```
و سپس باز کردن `http://localhost:8000` در مرورگر.

## ساختار اصلی

- `@@###@aahash/`
  - `config.php` — تنظیمات
  - `bot.php` — منطق بات
  - `lib/` — کتابخانه‌ها
  - `index.html`, `inline.php` — صفحات ورودی
